# MiniSocial 📱

A simple social media web application built using **Node.js, Express, MongoDB, and EJS**, where users can create posts, like/unlike posts, and upload images.

---

## 🚀 Features

- User authentication (login/signup system)
- Create text posts
- Like / Unlike posts (toggle system)
- Display like count dynamically
- Upload images using Multer
- User profile page
- EJS templating for frontend rendering
- MongoDB database integration

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js
- **Frontend:** HTML, CSS (Tailwind), EJS
- **Database:** MongoDB + Mongoose
- **File Uploads:** Multer
- **Authentication:** JWT / Sessions (based on your setup)
